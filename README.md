# FaceTrack2D

A lightweight library for **face detection, facial feature recognition, and real-time tracking** in 2D images. Includes demo applications to showcase **face detection**, **landmark tracking**, and **interactive real-time face feature analysis**. Ideal for computer vision experiments, AI projects, and learning purposes.

---

## Features

- Detect faces in images and video streams  
- Track facial landmarks (eyes, nose, mouth, etc.)  
- Real-time 2D face tracking  
- Demo applications to test and explore features
